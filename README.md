Titanic EDA (Exploratory Data Analysis)

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python, Pandas, Matplotlib, and Seaborn. The analysis includes data cleaning, handling missing values, statistical summaries, and visualizations to identify patterns affecting passenger survival.

### Key Insights

* Most passengers were between 20–40 years old.
* Fare distribution is highly right-skewed with several outliers.
* More passengers died than survived.
* Female passengers had a significantly higher survival rate.
* First-class passengers were more likely to survive than lower-class passengers.
* Correlation analysis highlights relationships between numerical features and survival outcomes.

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

### Workflow

1. Load and inspect the dataset.
2. Handle missing values in Age, Embarked, and Cabin columns.
3. Generate descriptive statistics.
4. Create visualizations for age, fare, survival, gender, and passenger class.
5. Perform correlation analysis to uncover feature relationships.

This project demonstrates fundamental data cleaning, visualization, and exploratory analysis techniques commonly used in data science workflows.
